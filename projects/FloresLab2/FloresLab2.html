What is a table?
A table is data formulated into Rows and Columns
When to use <tr>, <th>, <td>, <thead>, <tfoot>, and <caption>
<tr> is used to define to a table row
<th> is used to define a header cell
<td> is used when define to a table cell
<thead> is used group header content in a table
<tfoot> is used group footer content in a table
<caption> is used to define a table caption
What is the difference between colspan and rowspan?
Colspan will span cells across specific columns and Rowspan will span cells across specific rows
What is the difference between cells spacing and padding?
Cell padding is the spacing between the edges and the data, while cell spacing is only the space between the cells
How to define table borders using CSS
In CSS to define table borders you would use the border property
How to control border widths using CSS
In CSS to control border widths you would use the border-width property
